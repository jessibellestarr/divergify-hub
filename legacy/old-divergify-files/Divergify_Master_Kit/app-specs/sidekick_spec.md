# Sidekick (MVP) Spec
- **Purpose:** Micro-coach bubble with alarms, Focus Bubble, Beat your baseline.
- **Core:** Offline-first task list, quick timers, dopamine confetti, streaks.
- **Privacy:** Local storage by default, optional encrypted sync.
- **Flows:** Add task → choose baseline estimate → start → complete → micro‑reward.
- **Alarms:** Background notifications with skip/snooze/“worth it?” prompt.
- **Tech:** React Native + Expo, or Capacitor + web.
