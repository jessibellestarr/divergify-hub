# Store Embed Options
## Ko‑fi
- Create your Ko‑fi Shop products.
- Copy your Shop URL and replace the link in `/web/store.html`.
- If Ko‑fi provides an embed, paste it where indicated in `store.html`.

## Printful / Printify
- Use their “embed” or “storefront” iframe if available.
- If no embed works, link out to the storefront URL. Keep it simple until v2.
